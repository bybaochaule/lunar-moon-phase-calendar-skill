# Moon Phase Calendar Builder

Moon Phase Calendar Builder is an agent skill that creates clear lunar moon phase calendars for a chosen year, date range, timezone, and use case.

It helps produce:

- Printable moon phase calendars
- Monthly lunar planner layouts
- New Moon, First Quarter, Full Moon, and Last Quarter phase tables
- Timezone-aware moon phase schedules
- CSV-ready or calendar-import-ready event lists
- Planning notes for astrology, gardening, journaling, wellness, classrooms, content planning, and personal use
- Quality checks for source notes, timezone labels, missing phases, formatting, and unsupported claims

The skill is designed to help users turn basic calendar details into a practical, organized lunar planning calendar. It does not invent moon phase dates, guarantee outcomes, or replace final source verification before publishing, selling, printing, or relying on the calendar.
